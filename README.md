# 🏠 Home Assistant Camera Snapshot Integration

## 📖 Description

**A comprehensive Home Assistant integration for camera  

**Key Features:**
- ✅ Image snapshot by url  
- ✅ Camera availabilty info
- ✅ Update interval in seconds

## 🔌 Supported Devices

**All device than can provide a jpeg image by url
